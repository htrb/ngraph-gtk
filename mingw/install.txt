How to make Ngraph-gtk on Windows with MinGW


1. Get and install MinGW

http://sourceforge.net/projects/mingw/files/

Download mingw-get-inst-20110316.exe
Execute mingw-get-inst-20110316.exe
Check "MSYS Basic System" and "MinGW Developer Toolkit"


2. Get GTK+

http://ftp.gnome.org/pub/gnome/binaries/win32/gtk+/2.22/

gtk+_2.22.1-1_win32.zip
gtk+-dev_2.22.1-1_win32.zip
gtk+-bundle_2.22.1-20101227_win32.zip


3. Install GTK+

extract all files in C:\mingw


4. Get readline libraries

http://sourceforge.net/projects/gnuwin32/files/readline/5.0-1/

readline-5.0-1-bin.zip
readline-5.0-1-lib.zip


5. Install readline libraries

extract all files in C:\mingw


6. Get GSL

http://www.gnu.org/software/gsl/
gsl-1.14.tar.gz


7. Compile and install GSL

tar xf gsl-1.15.tar.gz
cd gsl-1.15
./configure --prefix=/mingw
make
make install


8. Get Ngraph-gtk

http://sourceforge.net/projects/ngraph-gtk/files/
Ngraph-gtk-6.06.02.tar.gz


9. Compile and install Ngraph-gtk

tar xf Ngraph-gtk-6.06.02.tar.gz
cd Ngraph-gtk-6.06.02
sh mingw/windows.sh

Ngraph-gtk will be installed in C:\ngraph-gtk if there are no error
during the compilation.
