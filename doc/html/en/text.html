Control characters for text drawing


To insert control characters in the Legend text string in the Text dialog box, you can change in character format.

Basic
^ Superscript letter will follow
_ Subscript characters will follow
@ Superscript, subscript Clear
\B Subsequent characters are in bold which
\I Subsequent characters in italics becomes
\N Bold italic Clear
\. Equivalent to the space
\b Backspace (width 'h')
\n Break
\- Hyphen
\& Aligned to the position of \ and when multiple rows by newline \ n is text
\ % \ @ Character next time ^ _, and outputs them as control characters to deal


Font
%F{Alias name} To change the font alias names
%S{integer} To specify the font size in points
%P{integer} Character spacing specified in point
%X{integer} Specify the amount by which point the current point is moved to the right
%Y{integer} Amount specified in point move down the current point


Formulas
%[Expression] Value of the expression


Shaping
%pf{format value} format and output the value in the format
  • For more information, please see the format man 3 printf.
  • `* M $ 'or' * 'to the precision specification is not available.
  • The length modifiers hh, h, L, q, j, z, Z, t is not available.
  • F conversion specifier, a, A, m, n, p, C, S is not available.

For example:
%pf{%e 1} is 1.000000e+00
%pf{%2.2f %[exp(1)]} is 2.72
