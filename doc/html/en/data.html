The data file


Data file is a text file format is basically as follows.

Comments in the data file
  • Data file is the first part of the Data dialog box [Load ...] read together so as to skip a specified number of lines in the first few lines of skip, etc. You can leave your comments.
  • By default, this line is the line number 0.
  • Comments may or may not.

Way to insert comments into the file
  • For comments on the way the Data File dialog box to the beginning [Load ...] the letter that is set in Remark.
  • The default is% # '.

Reading from data file
  • Data dialog box for you to write data files embedded option in the first row of the data file [Load settings] can read data files from various settings, press the button.

Data
  • Column 1, column 2, column 3, ... put the numeric data.
  • The maximum number of columns of data files is 99.
  • There is no maximum line length limitation.
  • Data dialog box, the column separator [Load ...] You can set the IFS.
  • By default, spaces, commas, tabs, and parentheses.
  • Carriage return is ASCII code (0A) 16. (0D) 16 are ignored.

Exponent of the numerical data given
  • The exponent of the numerical data specifiers e, E, you can use the D.
  • Cases) 1.0D+002

Handling of empty lines
  • Null line (line only IFS) is skipped.

Shows how to specify the missing data in Ngraph. For example, consider the case where data is missing in the second row.

Diagonal plot data format, data files of the rectangle


Diagonal form of data plots, rectangle, rectangle_fill, data file format used when you want to plot a rectangle and diagonals and using rectangle_solid_fill.

Example: (0, 0) - (1, 1) and (1, 1) - (1, 2) diagonal to create a file like this: draw a rectangle or time, (X) 1 a column, (Y) to 3 column.

Point X and Y, Y and X of the end point must be continuous, respectively. (Need not be consecutive four)


 

Error_bar_x plot data format data files


You can plot the error bars plotted for the data format error_bar_x.

For example, consider the following data.

X Error (X) Y
1.0 + / - 0.5 2.0
2.0 + / - 1.0 3.0


In this case, create a data file as follows (X) 1 the column, (Y) to 4 column.

1.0 -0.5 0.5 2.0
2.0 -1.0 1.0 3.0


X, the lower the error of X, the maximum error of three columns of X must be contiguous.
